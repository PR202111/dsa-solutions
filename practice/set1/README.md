# Practice Set 1: Fundamental Problems

## Overview
20 practice problems covering fundamental data structures and algorithms concepts. These problems are designed to build strong foundations in programming.

## Problem Categories

### Basic Data Structures (ques1-5)
- Array operations
- String manipulation  
- Simple searching

### Intermediate Problems (ques6-15)
- Sorting algorithms
- Linked lists
- Basic tree operations

### Advanced Concepts (ques16-20)
- Stack and queue applications
- Graph basics
- Recursion problems
