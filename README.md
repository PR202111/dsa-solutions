## 🚀 Getting Started

### Prerequisites
- GCC compiler (or any C compiler)
- Basic C programming knowledge

### Compilation

To compile a C file:
```bash
gcc -o output_file source_file.c
```

Example:
```bash
cd assignment/assignment1
gcc -o solution solution.c
./solution
```

## 📝 How to Use

1. Navigate to the relevant folder (assignment or practice set)
2. Open the C file in your editor
3. Compile using the instructions above
4. Run the executable
