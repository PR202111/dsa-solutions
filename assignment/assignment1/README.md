# Assignment 1: Tree Traversal & BST Operations

## Overview
Solutions for tree traversal problems, including Binary Search Tree (BST) operations and various traversal techniques like zigzag traversal.

## Files
- `solution1.c` - BST Zigzag Traversal
- `solution2.c` - Tree Operations

## Topics Covered
- Binary Search Trees (BST)
- Tree Traversal (In-order, Pre-order, Post-order)
- Level Order Traversal
- Zigzag Traversal
- Tree Insertion and Search

## Compilation & Execution
```bash
gcc -o solution1 solution1.c
./solution1
```

## Problem Description
These solutions demonstrate:
1. How to perform zigzag traversal on a BST
2. Managing nodes using arrays/stacks for level-order processing
3. Alternating direction traversal patterns
